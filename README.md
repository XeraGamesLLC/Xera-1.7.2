## Features

- **Combat** — Aura, AutoCrystal, Criticals, Reach
- **Movement** — Fly, Speed, NoFall, Jesus, Phase
- **Exploits** — PacketCancel, XCarry, FastBow, NoRotate
- **Visuals** — ESP, Chams, Tracers, XRay, Shaders
- **World** — Nuker, Scaffold, AutoTool, AutoHighway
- **Misc** — Spammer, AutoEat, DiscordRPC, Translator

Bundled with **Optifine HD_U F7** and **World Downloader**.
